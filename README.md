# EBookRentSystem
**21학년도 1학기 데이터베이스 과목에서 진행한 텀프로젝트 입니다.**  
온라인 도서 대출 관리 시스템을 주제로 개발하였습니다.

# 기술 스택
## 프론트엔드
- JavaScript
- JQuery
- Html
- Css
- BootStrap


## 백엔드
- OracleSQL
- PHP

# 구현기능

## 일반 회원
- 로그인 및 회원가입
- 도서 대출 및 예약
- 대출 가능 도서 목록 조회
- 도서 목록 검색
- 도서 상세정보 조회
- 현재 본인이 대출/예약 중인 도서 확인
- 예약 중인 도서 예약 취소
- 대출 도서 연장

## 관리자
- 새로운 도서 추가
- 현재 대출중인 도서 정보 조회
- 도서별 대출 횟수 통계
- 회원별 대출 도서 조회
